# python-revit-ifc-bim
Using Python to interact with Revit API, IFC files &amp; BIM workflows

### Setup & Requirements
Create virtuall environment & install `ifcopenshell`
```
python3 -m venv .venv
source .venv/bin/activate
pip install ifcopenshell
```

Dependencies
```

isodate==0.7.2

numpy==2.3.4
```
